# Symulator Wycinarki Plazmowej

Projekt zaliczeniowy – symulacja robota kartezjanskiego (wycinarki plazmowej).

## Uruchomienie
```bash
pip install -r requirements.txt
python main.py
```

## Sterowanie (tryb manualny)
- Strzalki → ruch glowicy
- Spacja → wlacz/wylacz plazme
- R → nagraj sciezke
- P → odtworz nagrana sciezke
